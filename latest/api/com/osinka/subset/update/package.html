package update

Update Operations

Basically, it is possible to create update operators and compose them. See Update and Modifications

Subset

Subset's updateMatch method allows to create an "positional" update operation relative to some enclosing sub-document. The example demonstrates this:

collection.update(BlogPost.Comments.by === "joe", BlogPost.Comments.updateMatch {_.votes inc 1})

This is equivalent to MongoDB shell

db.collection.update({"comments.by": "joe"}, {$inc: {"comments.$.votes": 1}})
