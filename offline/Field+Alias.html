Subset — Field Alias

Field Alias

Sometimes a subset’s field can be used so frequently, that it makes sense to create a field alias. The idea is to avoid repetiting code like 

val query = subset.where{_.field === 10}

and instead write 

val alias = field.in(subset)
val query = alias === 10
