Update Operations

Update operators and their composition. SeeUpdate and Modifications

Subset

Subset's matched method lets create a "positional" update modifier relative to the enclosing sub-document.

collection.update(BlogPost.comments.where{_.by === "joe"}, BlogPost.comments.matched.modify {_.votes inc 1})

This is equivalent to MongoDB shell

db.collection.update({"comments.by": "joe"}, {\$inc: {"comments.$.votes": 1}})
